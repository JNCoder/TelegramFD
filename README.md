## 功能
### 1、获取消息链接
> 在Telegram的私人聊天对话中、频道中或者群组中，找到你需要下载的媒体（视频、图片、ipa、txt、apk、音频等等），右键单击媒体文件选择 `Copy Message Link`

### 2、下载指定消息链接
> 如下图所示，在左边的菜单栏中，选择`下载指定消息`，然后在输入框中粘贴消息链接

![下载指定消息链接](https://www.telegramfd.site/assets/img/github/download-url.gif)

### 3、下载指定范围的消息
> 如下图所示，在左边的菜单栏中，选择`下载指定范围的消息`，注意`开始链接`与`结束链接`必须来自于同一个频道或者群组；并且`开始链接后面的数字要小于结束链接后面的数字`

![下载指定范围的消息](https://www.telegramfd.site/assets/img/github/download-range.gif)

### 4、下载指定频道或群组的全部消息
> 如下图所示，先复制需要下载的频道或者群组的分享链接，粘贴到输入框后，点击`下载`

![下载指定频道或群组的全部消息](https://www.telegramfd.site/assets/img/github/download-channel.gif)

## 设置
![设置](https://www.telegramfd.site/assets/img/github/setting.jpg)
- **线程数**
> 将大文件流分割成很多个小份，开启”线程数”个线程同时进行下载，以此提高下载速度；默认5个

- **文件类型**
> 指定下载的文件类型，目前只支持”全部”、”图片”以及”视频”；以后还会增加更多；默认全部

- **抓取文本**
> 将消息中的文字保存为同名txt文件；默认不勾选

- **自动下载评论消息**
> 当下载到此消息，如果勾选了此选项，则会继续下载12条评论中包含的所有媒体文件；如果没有勾选此选项，则只会下载这条消息的图片文件，而评论中的媒体文件则不会下载；默认勾选

- **下载指定频道或群组时，每次限制数**
> 如果一次性下载整个频道或者群组的全部消息，那需要的时间太长了，并且也不利于保全电报帐号。所以特设置此选项，表示每一次下载多少条消息，下次下载时，则接着上次的消息继续下载，默认值为20

- **代理设置：目前只支持socks协议**
> 服务器：一般都是指本机127.0.0.1
端口：请点击查看默认端口，如果还无法解决请联系客服

## 常见问题列表
- **找不到下载的文件：**
> 默认保存在软件的根目录下，用户可自行到`设置`窗口进行修改；

- **无法登陆：**
> 在**设置**窗口中输入"代理软件"的`服务器地址`与`端口`(socks协议)；
服务器：`127.0.0.1`
端口：每个代理软件都有所不同

| 名称 | 端口 |
| --- | --- |
|    Clash / Clash Verge / Meta   |  7890（SOCKS5）7897（HTTP）     |
|   V2RayN / V2RayU    |   1080    |
|    Shadowsocks / SSR   |   1080    |
|    Surge   |   用户自定义    |
